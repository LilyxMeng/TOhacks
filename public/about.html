COINSPIRE

Graphic Art Just Got Better!

COINSPIRE is a collaborative place where users from around the world can contribute a single pixel that will end up being a graphic art!


Inspiration

COINSPIRE is a collaborative place where users from around the world can contribute a single pixel that will collaboratively build up into elaborate graphic art! With a timeout of 1 second, users cannot get very far working on their own, and are forced to collaborate to take over the board.

What It Does

COINSPIRE is a collaborative place where users from around the world can contribute a single pixel that will collaboratively build up into elaborate graphic art! With a timeout of 1 second, users cannot get very far working on their own, and are forced to collaborate to take over the board.

How We Built It

We built COINSPIRE with HTML, CSS with bootstrap, and JavaScript for our frontend, Node.js for our backend, socket.io for webhooks and real-time client-server communication, and SQL with Cockroach db for our database to store all data received to transmit updates in live time to all users. We hosted the website on google cloud services and reserved the coinspire.tech domain on domain.com.

Challenges We Ran Into

This was our team's first time working with professional database services as well as implementing features such as live time updates. We faced a lot of difficulty reading a writing to our database, as we cannot have duplicate (x, y) coordinates and reserve compute time.

What We Learned

We learned how to access and use databases, write SQL, communicate with websockets, host dynamic websites via cloud computing, and work together as a team. Additionally, we have also learned some essential skills like time management skills and collaborating.

Whats Next for Coinspire?

We plan to expand our project and add more features like converting the canvas to a PNG file and sharing options to make it more interactive. Furthermore, we plan to add collaborative rooms in the server where a group of users can create their own unique art. We are also planning to add more drawing tools, like a paintbrush and such as multiple canvas.

How Does Our Project Help?

After the covid-19 pandemic, our team was feeling that the world was a bit more bland. We wanted a way for people to come together to create something that would benefit everyone while being a fun and somewhat rewarding experience. We found that the solution to this problem was to build a piece of living art, one that evolves with the people contributing to it.
