How To Use Coinspire?


In order to start drawing in Coinspire, you will need to be in this page, and you can select the color you want to draw with. By default, the blue color has already been selected. In order to change color, you can click on the top right button and select the color you want. Furthermore, you can also change the color based on the RGB value. To change the color based on the RGB value, you would need to follow the same steps, which is to click on the color button, and enter the RBG value.


Though we have not impletemented the undo action, we are working on it in the future update. But for now, if you would like to erase something, you would have to select a black color as it matches with the background, that way it will look like nothing ever happened at that pixel as it will overwrite it.
